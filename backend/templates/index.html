Routes

  1. /{source_id}/search/{search_keyword} : For searching a manga
  2. /{source_id}/manga?url={manga_url} : For fetching manga details
  3. /{source_id}/chapter?url={chapter_url} : For fetching chapter data

Important!

General: You can only use urls of mangas/chapters etc. same as the source. The source id helps in looking up which source we are going to scrape. The url and source MUST MATCH!

Pages: The page links won't work by directly opening them. You have to pass a referer header. The value of this header is the source url for example: https://readmangabat.com/ if you are using source id 1 (i.e. Mangabat).
